Comment ça fonctionne

Voici comment obtenir une carte et l'utiliser :

  1. Choisissez une carte : Sélectionnez la carte qui vous convient parmi nos options.
  2. Mode de paiement : Choisissez un mode de paiement parmi :
    • Ticket de recharge PCS ✅💯
    • Ticket de recharge Neosurf ✅💯
    • Cryptomonnaie Bitcoin (BTC) ✅💯
  3. Livraison : Fournissez les informations suivantes après le paiement :
    • ✔️ Votre Nom
    • ✔️ Votre Prénom
    • ✔️ Adresse complète
    • ✔️ Code postal, ville et pays

⭐️ Comment se passe l'envoi :

  • ⚡️ Une fois la commande effectuée, nous vous envoyons :
  • ®️ Une preuve d'envoi
  • ®️ Un numéro de suivi
  • ®️ Une preuve de mise en courrier

Livraison sous 24h à 48h 💫

Comment effectuer le retrait :

  • 💢 Insérez la carte dans le distributeur.
  • 💢 Tapez le code de retrait à 4 chiffres inscrit sur la carte.
  • 💢 Vous pouvez retirer jusqu'à 1000 € par retrait.

Après réception de votre carte, veuillez envoyer une photo ou une vidéo de satisfaction. Merci !
